# alexko

`npx alexko` — brand alias of the **same CLI** as [`ruflo`](https://www.npmjs.com/package/ruflo):
the AlexKo AI agent orchestration platform (a thin wrapper around `@claude-flow/cli`).

```bash
npx alexko init --wizard
npx alexko --help
```

`npx ruflo` continues to work and runs the identical CLI. See the repository README for full docs:
https://github.com/sashakobtsev21-stack/my_agents
